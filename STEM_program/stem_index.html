How many tricycles can you make?

You can find out how many tricycles you can make as if you are solving a chemical equation!
If you input the numbers of frames (F), seats (S), handle bars (H), wheels (W) and pedals (P), you can get the number of tricycles you can make.
To make a tricycle, you need:

F + S + H + 3W + 2P
